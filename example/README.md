# Photo Opener Example

This is a complete example application demonstrating the features of the `photo_opener_view` package.

## Features Demonstrated

*   **Single Image Viewer**: Opening images with hero animations.
*   **Image Gallery**: Swipeable gallery with thumbnails and captions.
*   **Video Player**: Video playback with custom controls.
*   **Styles**: Switching between different visual styles (Modern, Minimal, etc.).
*   **Customization**: Examples of custom callbacks for share, download, etc.

## Running the Example

1.  `cd example`
2.  `flutter run`
